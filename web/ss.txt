История должна быть разведена индивидуально по каждому пользователю, Люси просматривает историю по пользователям, Стив только свою.

1. Таска получена Люси (Obtained)

2. Таска выполнена Люси (Complete)

3. Таска рестартнута Люси (Restart)

4. Сделано предложение Стиву по работе над таской (Make an offer)

5. Отменено предложение Стиву на работу над таской (Cancel an offer)

6. Предложение по работе над таской принято Сивом (Accept)

7. Стиву отказано в работе над таской (Reject delegation)

8. Подверждена работа над таской Люси (Accept delegation)

9. Внесены средства для оплаты работы по таске (Fund)

10. Подтверждено окончание работ по таске Стивом (Submit)

11. Работа Стива по таске принята Люси (Complete)

12. Оплата за работу по таске получена Стивом (Payment)
